# Redis安装

## 一、下载

[Redis官网](https://redis.io/)

[Redis5.0.14.1](https://github.com/tporadowski/redis/releases)

## 二、安装解压版

1. 解压到文件夹

2. 开机启动服务

   ```shell
   redis-server redis.windows.conft
   ```
